## Примеры

### Размер бейджа
Размер задается с помощью свойства `size`. Возможные значения свойтсва: `"l"` или `"s"`.

```tsx live
import React from 'react';
import { Badge } from '@sberdevices/plasma-ui';

export function App() {
    return (
        <div>
            <Badge text="Бейдж" size="l" />
            <Badge text="Бейдж" size="s" />
        </div>
    );
}
```

### Вид бейджа
Вид бейджа задается с помощью свойства `view`. Возможные значения свойства `view`:

+ `"primary"` – основной бейдж;
+ `"secondary"` – вторичный бейдж.

```tsx live
import React from 'react';
import { Badge } from '@sberdevices/plasma-ui';

export function App() {
    return (
        <div>
            <Badge text="Бейдж" size="l" view="primary" />
            <Badge text="Бейдж" size="l" view="secondary" />
        </div>
    );
}
```

### Иконка слева
В левой части бейджа можно отобразить иконку.
Если не указывать текст, бейдж будет содержать только заданную иконку:

```tsx live
import React from 'react';
import { Badge } from '@sberdevices/plasma-ui';
import { IconDownload } from '@sberdevices/plasma-icons';

export function App() {
    return (
        <div>
            <Badge text="Бейдж" size="l" view="primary" contentLeft={<IconDownload color="inherit" size="xs" />} />
            <Badge size="l" view="primary" contentLeft={<IconDownload color="inherit" size="xs" />} />
        </div>
    );
}
```
