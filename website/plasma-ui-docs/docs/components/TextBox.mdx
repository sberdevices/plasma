## Примеры

### Базовое использование

```tsx live
import React from 'react';
import { TextBox } from '@sberdevices/plasma-ui';

export function App() {
    return (
        <TextBox title="Заголовок" subTitle="Подзаголовок" />
    );
}
```

### Собственная сборка

```tsx live
import React from 'react';
import { TextBox, TextBoxTitle, TextBoxSubTitle } from '@sberdevices/plasma-ui';

export function App() {
    return (
        <TextBox>
            <TextBoxTitle>Заголовок</TextBoxTitle>
            <TextBoxSubTitle>Подзаголовок</TextBoxSubTitle>
        </TextBox>
    );
}
```
