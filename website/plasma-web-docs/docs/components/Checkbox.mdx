## Использование
Компонент `Checkbox` может содержать лейбл и описание.

```tsx live
import React from 'react';
import { Checkbox } from '@sberdevices/plasma-web';

export function App() {
    return (
        <Checkbox label="Чекбокс" description="Описание" defaultChecked />
    );
}
```
