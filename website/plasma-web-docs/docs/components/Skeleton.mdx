# Skeleton
Набор компонентов для создания скелетов загрузки (плейсхолдеров).

## LineSkeleton
<Description name="LineSkeleton" />
<PropsTable name="LineSkeleton" />

```tsx live
import React from 'react';
import { LineSkeleton } from '@sberdevices/plasma-web';

export function App() {
    return (
        <LineSkeleton size="headline3" />
    );
}
```

## На примере карточки

```tsx live
import React from 'react';
import { Button, withSkeleton } from '@sberdevices/plasma-web';

export function App() {
    const ButtonSkeleton = withSkeleton(Button);

    return (
        <ButtonSkeleton
            fullWidth
            text="Загрузка..."
            view="primary"
            size="s"
            scaleOnInteraction={false}
            outlined={false}
            skeleton={true}
            tabIndex={-1}
        />
    );
}
```
