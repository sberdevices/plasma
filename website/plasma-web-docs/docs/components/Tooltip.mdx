## Базовое применение

`Tooltip` оборачивает собой любой компонент и открывается в заданную сторону.

Предполагается оборачивать иконки из пакета `@sberdevices/plasma-icons`

```tsx live
import React from 'react';
import { Tooltip } from '@sberdevices/plasma-web';
import { IconApps } from '@sberdevices/plasma-icons';

export function App() {
    return (
        <div style={{marginLeft: '100px',marginBottom: '35px'}}>
            <Tooltip text="Высокое качество воспроизведения" visible>
                <IconApps />
            </Tooltip>
        </div>
    );
}
```
