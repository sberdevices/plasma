## Использование
Компонент `Radiobox` может содержать лейбл и описание.

```tsx live
import React from 'react';
import { Radiobox } from '@sberdevices/plasma-web';

export function App() {
    return (
        <Radiobox label="Радиокнопка" description="Описание" defaultChecked />
    );
}
```
