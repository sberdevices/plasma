# Typography
В состав Plasma Web входят компоненты, реализующие типографическую систему [типографическую систему Plasma](../design/typography).
Они заменяют обычные HTML-теги, такие как `h1`, `h2`, `h3`, `h4`, `p` и некоторые другие.

## Установка шрифта
Компоненты типографики используют фирменный шрифт **SB Sans Text**.
О том, как установить данный шрифт, смотрите на странице [Типографическая система](../design/typography#установка-шрифта).

## Отступы
По умолчанию типографика поддерживает стандартные отступы.
О том, как использовать отступы, читайте на странице [Стандартные отступы](../design/spacing#встроенные-отступы-в-типографике).

## Состав

### Заголовки

```tsx live
import React from 'react';
import { Headline1, Headline2, Headline3, Headline4 } from '@sberdevices/plasma-web';

export function App() {
    return (
        <div>
            <Headline1>Headline1</Headline1>
            <Headline2>Headline2</Headline2>
            <Headline3>Headline3</Headline3>
            <Headline4>Headline4</Headline4>
        </div>
    );
}
```

### Основной текст

```tsx live
import React from 'react';
import { Body1, Body2, ParagraphText1, ParagraphText2 } from '@sberdevices/plasma-web';

export function App() {
    return (
        <div>
            <Body1>Body1</Body1>
            <Body2>Body2</Body2>
            <ParagraphText1>ParagraphText1</ParagraphText1>
            <ParagraphText2>ParagraphText2</ParagraphText2>
        </div>
    );
}
```

### Вспомогательный текст

```tsx live
import React from 'react';
import { Footnote1, Footnote2 } from '@sberdevices/plasma-web';

export function App() {
    return (
        <div>
            <Footnote1>Footnote1</Footnote1>
            <Footnote2>Footnote2</Footnote2>
        </div>
    );

```
