## Хуки

### useCart
Позволяет получить и изменить состояние корзины. Для использования хука необходимо обернуть дерево компонентов компонентом `CartProvider`.

## Примеры

```tsx
import React from 'react';
import { Cart, CartProvider, EmptyCart } from '@sberdevices/plasma-temple';

export function App() {
    return (
        <CartProvider>
            <Cart
                onCheckout={() => {}}
                emptyCart={<EmptyCart imageSrc="/images/img.png" onGoToCatalog={() => {}} />}
            />
        </CartProvider>
    );
}
```
