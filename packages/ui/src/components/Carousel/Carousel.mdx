# Carousel

Набор компонентов для создания списков с прокруткой.
Для использования каждый дочерний элемент списка должен использовать CarosuselContext для регистрации ``ref`` – это необходимо для рассчёта положения элементов на экране (см. CarouselItem).

## Carousel
Основной компонент карусели.

### Props
<Props of={Carousel} />

## CarouselWrapper
Обертка над ``Carousel`` для компенсации отрицательных отступов.

## CarouselItem
Компонент для создания обычных каруселей.
Использует контекст ``CarouselContext`` для регистрации своего ``ref``.

### Props
<Props of={CarouselItem} />

## CarouselCol
Компонент, основанный на Grid/Col и предназначенный для использования в каруселях с сеткой.
Так же как и ``CarouselItem`` использует контекст ``CarouselContext``.

### Props
<Props of={CarouselCol} />

## Пример
<Playground>
    {() => {
        /**
         * В данном примере используются Grid и CSS Scroll Snap.
         * В корне приложения необходимо разместить DeviceThemeProvider.
         * В качестве основного блока лэйаута используется Container.
         * Опция inContainer у CarouselWrapper - необходима для того, чтобы компенсировать отступы контейнера.
         * Колонки (Col) помещаются в строки (Row).
         */
        return (
            <DeviceThemeProvider detectDeviceCallback={() => 'sberBox'}>
                <ThemeBackground>
                    <Container>
                        <CarouselWrapper inContainer>
                            <Row>
                                <Carousel
                                    axis="x"
                                    index={0}
                                    scrollSnap={true}
                                    scrollSnapType="mandatory"
                                    scrollSnapAlign="start"
                                >
                                    {Array(12)
                                        .fill(0)
                                        .map((item, i) => (
                                            <CarouselCol key={`item:${i}`} size={2} sizeM={1.5}>
                                                <Card roundness={12}>
                                                    <CardBody>
                                                        <CardMedia src={image} ratio="1:1" />
                                                    </CardBody>
                                                </Card>
                                                <Body1 style={{ marginTop: '0.75rem' }}>Song #81</Body1>
                                                <Footnote1 style={{ marginTop: '0.25rem' }}>
                                                    Untitled artist
                                                </Footnote1>
                                            </CarouselCol>
                                        ))
                                    }
                                </Carousel>
                            </Row>
                        </CarouselWrapper>
                    </Container>
                </ThemeBackground>
            </DeviceThemeProvider>
        );
    }}
</Playground>
