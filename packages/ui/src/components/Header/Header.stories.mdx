## Использование
> Используйте или скомпанованный компонент ``<Header />``,
или составные части, где корневым узлом будет являться ``<HeaderRoot />``.

> Шапка всегда используется там, где на странице присутствует ``<Container />``.

```tsx
// App.tsx
import { Container, Header }  from '@sberdevices/ui';

export const App = () => (
    <Container>
        <Header
            back={true}
            logo="./images/logo.png"
            logoAlt="Logo"
            title="Header title text"
            subtitle="Subtitle text"
            onBackClick={() => console.log('Back click!')}
        >
            <div>Header content</div>
        </Header>
    </Container>
);
```

## Комбинация свойств шапки
Шапка имеет свойства, которые недопустимо использовать без соответствующих им парных свойств:

```tsx
// Неправильно:
const a = <Header subtitle="Subtitle" />;
// Правильно:
const b = <Header title="Title" />;
const c = <Header title="Title" subtitle="Subtitle" />;

// Неправильно:
const d = <Header onBackClick={() => alert('Back click.')} />;
// Правильно:
const e = <Header back onBackClick={() => alert('Back click.')} />;

// Неправильно:
const f = <Header logoAlt="Fancy image." />;
// Правильно:
const g = <Header logo="/path/to/fancy-image.jpg" logoAlt="Fancy image." />;
```
