# Skeleton
Набор компонентов для создания скелетов загрузки (плейсхолдеров).

## LineSkeleton
Размеры компонента задаются с помощью констант и соответствуют размерам [типографических элементов](/typography).

> Используйте компонент, когда необходимо нарисовать отдельную строку текста с заданным размером.

### Пример
<Playground>
    <ThemeBackground>
        <LineSkeleton gradient size="headline3" />
    </ThemeBackground>
</Playground>

### Props
<Props of={LineSkeleton} />

## TextSkeleton
Компонент для создания плейсхолдера нескольких строк текста. Размеры компонента задаются с помощью констант и соответствуют размерам [типографических элементов](/typography).

> Используйте компонент, когда необходимо отобразить подряд несколько строк текста одного размера.

### Пример
<Playground>
    <ThemeBackground>
        <TextSkeleton gradient size="body1" lines={3} />
    </ThemeBackground>
</Playground>

### Props
<Props of={TextSkeleton} />

## RectSkeleton
Компонент для создания прямоугольного плейсхолдера.

> Используйте в случаях, когда необходимо нарисовать блок произвольной высоты и ширины, например плейсхолдер картинки.

### Пример
<Playground>
    <ThemeBackground>
        <RectSkeleton gradient width="2rem" height="2rem" />
    </ThemeBackground>
</Playground>

### Props
<Props of={RectSkeleton} />
