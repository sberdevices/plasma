# Типографическая система

## Примеры использования

Как компонент:

```jsx
import React from 'react';
import { Body1 } from '@sberdevices/ui/components/Typography';

const App = () => (
  <Body1>some text</Body1>
);

export default App;
```

Как объект стиля:

```jsx
import React from 'react';
import { body1 } from '@sberdevices/plasma-tokens';

const App = () => (
  <div style={ body1 }>some text</div>
);

export default App;
```
