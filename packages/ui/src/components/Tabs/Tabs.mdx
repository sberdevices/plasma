# Tabs
Набор компонентов для создания табов.
Для создания табов используется структура схожая со структурой маркированного списка, создаваемой с помощью тегов ``ul`` и ``li`` в HTML.
<ThemeBackground showcase>
    <Tabs>
        <TabItem isActive={true}>Элемент 1</TabItem>
        <TabItem>Элемент 2</TabItem>
        <TabItem>Элемент 3</TabItem>
        <TabItem>Элемент 4</TabItem>
    </Tabs>
</ThemeBackground>

## Tabs
Основной компонент, контейнер табов.

### Props
<Props of={Tabs} />

## TabItem
Элемент списка, недопустимо импользовать вне компонента ``Tabs``.

### Props
<Props of={TabItem} />

## Примеры использования

### Размер табов
Чтобы изменить размер, установите свойство ``size`` на компоненте ``Tabs`` в одно из следующих значений: ``"l"``, ``"m"``, ``"s"``.

``"secondary"``, ``"index"``, ``"clear"``.

### Иконки
Для создания квадратных табов с иконкой или текстом используйте свойство ``contentLeft``, передавая туда рендер иконки или текста.
Также, иконку можно помещать слева и справа от текста.
<Playground>
  <ThemeBackground>
    <Tabs>
        <TabItem isActive={true} contentLeft={<Icon icon="clock" size="s" />} />
        <TabItem contentLeft={<Icon icon="settings" size="s" />} />
        <TabItem contentLeft={<Icon icon="house" size="s" />} />
        <TabItem contentLeft={<Icon icon="trash" size="s" />} />
    </Tabs>
  </ThemeBackground>
</Playground>

### Модификаторы на компоненте ``Tabs``
* ``disabled`` — делает табы недоступными для нажатия;
* ``fixedWidth`` — кнопки табов примут фиксированную ширину, максимально равную 25% контейнера Tabs, при этом сам контейнер займет 100% ширины;
* ``outlined`` — добавляет рамку при фокусе;
* ``scaleOnInteraction`` — увеличение/уменьшение по ховеру/нажатию;
* ``pilled`` — кнопки табов и контейнер примут вид скругленных "капсул".
