# Card
Набор компонентов для создания карточек.

## Card
Контейнер со скругленными углами с возможностью фокусировки на нем.

### Props
<Props of={Card} />

## CardBody
Используйте этот компонент для корректного отображения скругленных углов контента.

### Props
<Props of={CardBody} />

## CardContent
Компонент для размещения в нем текстового или любого другого контента.

### Props
<Props of={CardContent} />

## CardMedia
Компонент для отображения картинок.

### Props
<Props of={CardMedia} />

## CardHeadline1, ...
``... CardHeadline2, CardHeadline3, CardFootnote1, CardFootnote2`` — типографика, поддерживающая изменение цвета через ``view``.

### Props
<Props of={CardHeadline1} />

## CardParagraph1, ...
``... CardParagraph2, CardBody1, CardBody2`` — типографика, с возможностью задания количества строк через ``lines`` и поддерживающая изменение цвета через ``view``.

### Props
<Props of={CardParagraph1} />

## Примеры использования

### Basic / value

Карточка информативного ответа. Может быть нескольких видов и представляет собой класс карточек. Может иметь ячейку-изображение.

                    <CardHeadline1 style={{ marginTop: '0.75rem' }}>до 230 000 ₽</CardHeadline1>
                    <CardFootnote1 style={{ marginTop: '0.375rem' }} view="secondary">
                        На 18 месяцев, ставка 13,9%

                    <Button
                        text="Label"
                        view="primary"
                        size="s"
                        scaleOnFocus
                        fullWidth
                        style={{ marginTop: '1em' }}
                        tabIndex={-1}
                    />
                </CardContent>
            </CardBody>
        </Card>
    </ThemeBackground>
</Playground>

### List card

Карточка-список / конструктор целого класса карточек, состоящих из списка ячеек (Cells).
Может быть представлена в нескольки вариантах.
По сути представляет собой набор стилей и параметров для обертки списка ячеек в карточку.
