# Stepper
Набор компонентов для создания контролов управления количеством, суммой и прочими численными показателями.

## Stepper
Готовый компонент для создания счетчика, подобного input[type="range"].

### Props
<Props of={Stepper} />

## StepperButton
Кнопка, позволяющая создавать контролы +/- и имеющая API как у Button.

### Props
<Props of={StepperButton} />

## StepperValue
Компонент для отображения значения.

### Props
<Props of={StepperValue} />

## StepperRoot
Стилизованный div, обертка над StepperButton и StepperValue.

### Props
Не имеет.

## Примеры использования

### Базовое применение
<Playground>
  <ThemeBackground>
    <Stepper
      value={5}
      step={1}
      min={0}
      max={10}
      disabled={false}
      onChange={() => {}}
    />
  </ThemeBackground>
</Playground>

### Кастомная сборка
