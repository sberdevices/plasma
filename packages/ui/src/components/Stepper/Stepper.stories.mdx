# Stepper
Набор компонентов для создания элементов управления количеством, суммой и другими численными показателями.

## Stepper
<Description of={Stepper} />
<Props of={Stepper} />

### Модификаторы
При использовании модификатора ``remover``, по достижению
минимального количества кнопка **минус** заменяется на **удалить**.
По нажатию на кнопку **удалить** вызывается обработчик **удаления**.

## Использование
Используйте или скомпанованный компонент ``<Stepper />``,
или составные части, где корневым узлом будет являться ``<StepperRoot />``.

```tsx
// App.tsx
import { Stepper }  from '@sberdevices/ui';

export const App = () => (
    <Stepper
      remover
      value={5}
      step={1}
      min={0}
      max={10}
      disabled={false}
      onChange={() => {}}
      onRemove={() => {}}
    />
);
```
