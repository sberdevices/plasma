# Grid
На каждом брейкпоинте разрешения свое максимальное количество столбцов.

## Container
Блок с полями по бокам для размещения контента по вертикали.
Недопустимо вкладывать этот блок сам в себя (или дальше по древу).

### Props
<Props of={Container} />

## Row
Блок с отрицательными отступами для размещения колонок (Col) по горизонтали. Недопустимо вкладывать этот блок сам в себя, но можно чередовать далее по древу с использованием Col.

### Props
<Props of={Row} />

## Col
Блок для размешения контента или строк (Row) внутри него.

### Col[type="rel"]
Поведение по умолчанию. У данного подтипа колонок ширина и отступ имеют относительное значение (в процентах).
Предназначено для верстки с страниц.

### Col[type="calc"]
У таких колонок ширина и отступ расчитываются на основе ширины контейнера и хранятся в css vars.
Предназначено для верстки каруселей. С примером использования можно ознакомиться в документации по [каруселям](/components/Carousel#пример).

### Props
<Props of={Col} />
