# Grid

## Container
Блок с полями по бокам для размещения контента по вертикали. Недопустимо вкладывать этот блок сам в себя (или дальше по древу).

### Props
<Props of={Container} />

## Row
Блок с отрицательными отступами для размещения ячеек (или столбцов, Col-s) по горизонтали. Недопустимо вкладывать этот блок сам в себя, но можно чередовать далее по древу с использованием Col.

### Props
<Props of={Row} />

## Col
Блок для размещения контента. На каждом брейкпоинте разрешения свое максимальное количество столбцов.

### Props
<Props of={Col} />
