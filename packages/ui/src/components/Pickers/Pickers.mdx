# Pickers
Набор контролов для выбора даты и времени.

## DatePicker
Компонент для выбора даты.

### Props
<Props of={DatePicker} />

### Пример
<Playground>
    <ThemeBackground>
        <DatePicker
            value={new Date(1980, 8, 1)}
            max={new Date(1985, 12, 31)}
            min={new Date(1975, 0, 1)}
            disabled={false}
            controls={false}
            visibleItems={5}
            onChange={(date) => console.log(date)}
        />
    </ThemeBackground>
</Playground>

## TimePicker
Компонент для выбора времени.

### Props
<Props of={TimePicker} />

### Пример
