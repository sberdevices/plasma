## Использование
Доступные иконки находятся в директории Icons.
Пример импорта одной иконки:
``` import { IconCart } from '@sberdevices/ui/components/Icon/Icons/IconCart' ```

## Как добавить новую иконку
1. Подготовить свг-разметку в виде реакт-компонента в директории Icon.assets
2. Собрать компонент иконки в директории Icons для отдельного использования
3. Импортировать свг в компонент Icon
