# Button
Компонент для отображения кнопки. Может быть отображена в нескольких размерах и цветах, с текстом и иконкой или без оных.

## Props
<Props of={Button} />

## Примеры использования

### Базовое применение
Допустимо указывать надпись через свойство ``text`` (в связке со свойствами ``contentLeft`` и ``contentRight``) или любой контент напрямую через ``children``
<Playground>
  <ThemeBackground>
    <Button text="Это текст" />
    {'\u00A0\u00A0\u00A0\u00A0'}
    <Button>
        <span>Это тоже текст</span>
    </Button>
  </ThemeBackground>
</Playground>

### Размер кнопки
Чтобы изменить размер кнопки, установите свойство ``size`` в одно из следующих значений: ``"l"``, ``"m"``, ``"s"``.
<Playground>
  <ThemeBackground>
    <Button text="Button" size="l" />
    {'\u00A0\u00A0\u00A0\u00A0'}
    <Button text="Button" size="m" />
    {'\u00A0\u00A0\u00A0\u00A0'}
    <Button text="Button" size="s" />
  </ThemeBackground>
</Playground>

### Вид кнопки
Чтобы изменить вид кнопки, установите свойство ``view`` в одно из следующих значений: ``"primary"``, ``"secondary"``, ``"warning"``, ``"critical"``, ``"checked"``, ``"clear"``.
<Playground>
  <ThemeBackground>
    <Button text="Button" view="primary" size="s" />
    {'\u00A0\u00A0\u00A0\u00A0'}
    <Button text="Button" view="secondary" size="s" />
    {'\u00A0\u00A0\u00A0\u00A0'}
    <Button text="Button" view="warning" size="s" />
    {'\u00A0\u00A0\u00A0\u00A0'}
    <Button text="Button" view="critical" size="s" />
    {'\u00A0\u00A0\u00A0\u00A0'}
    <Button text="Button" view="checked" size="s" />
    {'\u00A0\u00A0\u00A0\u00A0'}
    <Button text="Button" view="clear" size="s" />
  </ThemeBackground>
</Playground>

### Модификаторы
``disabled`` делает кнопку недоступной для нажатия, ``fullWidth`` позволяет ей занять доступню ширину, а с помощью ``motion`` можно управлять анимацией при наведении/нажатии.
<Playground>
  <ThemeBackground>
    <div style={{ display: 'flex', flexDirection: 'column', alignItems: 'flex-start', width: '100%', padding: 40 }}>
      <Button text="Недоступно" disabled />
      {'\u00A0\u00A0\u00A0\u00A0'}
      <Button text="Широкая кнопка" fullWidth />
      {'\u00A0\u00A0\u00A0\u00A0'}
      <Button text="Без анимации" motion={false} />
    </div>
  </ThemeBackground>
</Playground>

## ActionButton
Упрощенная версия Button для создания квадратных (со сторонами 1:1) кнопок, с меньшими, чем у Button, размерами.
