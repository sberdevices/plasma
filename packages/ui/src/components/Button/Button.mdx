# Button
Компонент для отображения кнопки. Может быть отображена в нескольких размерах и цветах, с текстом и иконкой или без оных.

## Props
<Props of={Button} />

## Примеры использования

### Базовое применение
Допустимо указывать надпись через свойство ``text`` (в связке со свойствами ``contentLeft`` и ``contentRight``) или любой контент напрямую через ``children``
<Playground>
  <ThemeBackground spaced>
    <Button text="Это текст" />
    <Button>
        <span>Это тоже текст</span>
    </Button>
  </ThemeBackground>
</Playground>

### Размер кнопки
Чтобы изменить размер кнопки, установите свойство ``size`` в одно из следующих значений: ``"l"``, ``"m"``, ``"s"``.
<Playground>
  <ThemeBackground spaced>
    <Button text="Button" size="l" />
    <Button text="Button" size="m" />
    <Button text="Button" size="s" />
  </ThemeBackground>
</Playground>

### Вид кнопки
Чтобы изменить вид кнопки, установите свойство ``view`` в одно из следующих значений: ``"primary"``, ``"secondary"``, ``"warning"``, ``"critical"``, ``"checked"``, ``"clear"``.
<Playground>
  <ThemeBackground spaced>
    <Button text="Button" view="primary" />
    <Button text="Button" view="secondary" />
    <Button text="Button" view="warning" />
    <Button text="Button" view="critical" />
    <Button text="Button" view="checked" />
    <Button text="Button" view="clear" />
  </ThemeBackground>
</Playground>

### Иконки
Для создания квадратных кнопок с иконкой или текстом используйте свойство ``contentLeft``, передавая туда рендер иконки или текста.
Также, иконку можно помещать слева и справа от текста.
<Playground>
  <ThemeBackground spaced>
    <Button contentLeft={<IconDownload />} />
    <Button contentLeft={42} />
    <Button text="Button" contentLeft={<IconDownload />} />
    <Button text="Button" contentRight={<IconDownload />} />
  </ThemeBackground>
</Playground>

### Модификаторы
``disabled`` делает кнопку недоступной для нажатия,
``fullWidth`` позволяет ей занять доступню ширину,
``outlined`` добавляет рамку при фокусе,
a с помощью ``motion`` можно управлять анимацией при наведении/нажатии.
<Playground>
  <ThemeBackground spaced>
    <Button text="Недоступно" disabled />
    <Button text="Широкая кнопка" fullWidth />
    <Button text="Без фокуса" outlined={false} />
    <Button text="Без анимации" motion={false} />
  </ThemeBackground>
</Playground>

## ActionButton
Упрощенная версия Button для создания квадратных (со сторонами 1:1) кнопок.
Размеры ActionButton меньше, чем у Button.
Цвета, размеры и модификаторы имеет те же, что у основной кнопки.

### Props
<Props of={ActionButton} />

### Примеры использования
<Playground>
  <ThemeBackground spaced>
    <ActionButton size="s">
      <IconDownload />
    </ActionButton>
    <ActionButton size="m">
      <IconDownload />
    </ActionButton>
    <ActionButton size="l">
      <IconDownload />
    </ActionButton>
  </ThemeBackground>
</Playground>
