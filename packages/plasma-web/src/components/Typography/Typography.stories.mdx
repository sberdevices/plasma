# Типографическая система

## Примеры использования

### Как компонент
```tsx
// App.tsx
import React from 'react';
import { Body1 } from '@sberdevices/plasma-web/components/Typography';

const App = () => (
  <Body1>some text</Body1>
);

export default App;
```

### Как объект стиля
```tsx
// App.tsx
import React from 'react';
import { body1 } from '@sberdevices/plasma-tokens-web';

const App = () => (
  <div style={ body1 }>some text</div>
);

export default App;
```
