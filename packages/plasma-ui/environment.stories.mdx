# Окружение

Смартапы запускаются внутри [WebView](https://webkit.org/) — компонента, отображающего веб-страницы внутри приложений.

## Способы взаимодействия с устройствами
Все устройства поддерживают голосовое управление с помощью [AssistantNavigationCommand](https://github.com/sberdevices/assistant-client#AssistantNavigationCommand).  
Кроме этого к устройствам можно подключиться с помощью приложения Сбер Салют и использовать его в качестве пульта. Управление устройствами с помощью Сбер Салют возможно как голосовыми командами, так и жестами.

Чтобы поддержать управление с помощью пульта [подпишитесь на события](https://github.com/sberdevices/assistant-client#%D0%BD%D0%B0%D0%B6%D0%B0%D1%82%D0%B8%D0%B5-%D0%BA%D0%BD%D0%BE%D0%BF%D0%BE%D0%BA-%D0%BD%D0%B0-%D0%BF%D1%83%D0%BB%D1%8C%D1%82%D0%B5) нажатия клавиш на клавиатуре и подключите библиотеку [spatial-navigation](https://github.com/sberdevices/spatial-navigation).

Устройства поддерживают следующие способы управления:
- SberBox - пульт;
- SberBox Top - пульт и жесты;
- SberPortal - сенсорный интерфейс, жесты, управление через Сбер Салют;
- СберБанк Онлайн / Сбер Салют - сенсорный интерфейс.

Для корректной навигации по экранам смартапа постройте дерево переходов с помощью [History API](https://github.com/sberdevices/assistant-client#%D0%BD%D0%B0%D0%B2%D0%B8%D0%B3%D0%B0%D1%86%D0%B8%D1%8F-%D0%BF%D0%BE-%D1%81%D1%82%D1%80%D0%B0%D0%BD%D0%B8%D1%86%D0%B0%D0%BC-%D1%81%D0%BC%D0%B0%D1%80%D1%82%D0%B0%D0%BF%D0%B0).

## Поверхности на которых загружается WebView

Для определения типа устройства используйте утилиту [detectDevice](/?path=/docs/core-utils--page#detectdevice).

Разрешения:  
- SberBox top - 1920x1080 (fullHD);
- SberBox - 1920x1080 (fullHD);
- SberPortal - 1280x800.

Инсеты:  
Панель асистента перекрывает часть WebView. Узнать параметры установленных инсетов можно с помощью действия insets ([AssistantInsetsCommand](https://github.com/sberdevices/assistant-client#AssistantInsetsCommand)), которое можно получить подписавшись на `assistantClient.onData()`.

## devicesScale и Сетки 

[Документация для компонентов сетки](/?path=/docs/layout-grid--default).

Ширина сетки на устройствах:
- SberPortal - 6 колонок;  
- SberBox - 12 колонок;  
- СберБанк Онлайн / Сбер Салют - 4 колонки.

Точка отсчета размерности — 16 пикселей.  
Для SberPortal и SberBox `deviceScale = 2`, то есть точка отсчета размерности — 32 пикселя.  

## Влияние персонажей на тему

Выбранный персонаж влияет на [цветовую палитру](/?path=/docs/colors--default) ассистента.  
При смене персонажа происходит `Assistant action` - `{ type: 'character'}`  
([AssistantCharacterCommand](https://github.com/sberdevices/assistant-client#AssistantCharacterCommand)). [Подробнее о смене персонажа](/?path=/docs/theming--page).    

В зависимости от выбранного персонажа, при инициализации webview устанавливается фон соответсвтующий фон:
- Сбер — градиент зеленого;
- Афина — градиент синего;
- Джой — градиент оранжевого.

Основные цвета персонажей:
- Сбер — зеленый;
- Афина — синий;
- Джой — фиолетовый.
