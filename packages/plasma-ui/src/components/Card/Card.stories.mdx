## CardHeadline1, ...

## CardParagraph1, ...
``... CardParagraph2, CardBody1, CardBody2`` — типографика, которой можно указать количество строк с помощью свойства ``lines`` и изменить цвет с помощью свойства ``view``.

## Использование

```tsx
// App.tsx
import {
    Card,
    CardBody,
    CardContent,
    CardMedia,
    CardHeadline1
 } from '@sberdevices/plasma-ui';

export const App = () =>  (
    <Card>
        <CardBody>
            <CardMedia src="/path-to-image.jpg" ratio="1 / 1" />
            <CardContent>
                <CardHeadline1>Hello, world!</CardHeadline1>
            </CardContent>
        </CardBody>
    </Card>
);
```
