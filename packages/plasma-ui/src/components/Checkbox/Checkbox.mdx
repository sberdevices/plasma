# Checkbox
Флажок или *чекбокс*. Позволяет пользователю управлять параметром с двумя состояниями — ☑ включено и ☐ отключено.

## Props
<Props of={Checkbox} />

## Пример
<Playground>
    {() => {
        const [checked, setChecked] = React.useState(false);
        return (
            <ThemeBackground>
                <Checkbox
                    id="agreement"
                    name="agreement"
                    value="yes"
                    label="Чекбокс"
                    checked={checked}
                    onChange={(event) => setChecked(event.target.checked)}
                />
            </ThemeBackground>
        );
    }}
</Playground>
