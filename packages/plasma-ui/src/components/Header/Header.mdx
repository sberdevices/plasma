# Header
Набор компонентов для создания шапки.

## Props
<Props of={Header} />

## Примеры

### Использование готовой шапки
Воспользовавшись готовым компонентом ``Header``, можно отобразить шапку, указав свойства ``title``, ``logo`` и пр.
<Playground>
    <ThemeBackground>
        <Container>
            <Header
                back
                logo={logo}
                logoAlt="Logo"
                title="Header title text"
                subtitle="Subtitle text"
            >
                <Button view="clear" shiftRight>Header button</Button>
            </Header>
      </Container>
    </ThemeBackground>
</Playground>

### Создание собственной шапки
Если возможностей компонента ``Header`` недостаточно, можно создать собственную шапку.
Используйте такие компоненты, как ``HeaderRoot``, ``HeaderTitle``, ``HeaderContent`` и другие.
<Playground>
    <ThemeBackground>
        <Container>
            <HeaderRoot>
                <HeaderBack />
                <HeaderLogo src={logo} alt="Logo" />
                <HeaderTitleWrapper>
                    <HeaderTitle>Header title text</HeaderTitle>
                    <HeaderSubtitle>Subtitle text</HeaderSubtitle>
                </HeaderTitleWrapper>
                <HeaderContent>
                    <Button view="clear" shiftRight>Header button</Button>
                </HeaderContent>
            </HeaderRoot>
        </Container>
    </ThemeBackground>
</Playground>
