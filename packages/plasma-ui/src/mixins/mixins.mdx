# Миксины (примеси)
Раздел содержит функции-примеси для компонентов, которые стилизуются с помощью 'styled-components'.

> Для удобства типизации, каждый модуль, в котором лежит миксин, дополнительно экспортирует соответствующий интерфейс ``props``.

## addFocus
Добавляет фокусную рамку компонентам, для которых возможно фокусирование или если передан флаг ``focused``.

### Props
<Props of={addFocus} />

### Использование
```tsx
import styled from 'styled-components';
import { FocusProps, addFocus } from '@sberdevices/ui/mixins';

// Способ 1. Передача props напрямую в вызываемый компонент
const Button1 = styled.button<FocusProps>`
    ${addFocus}
`;

<Button1
    outlined={true}
    outlineSize="2px"
    outlineOffset="2px"
    outlineColor="green"
    outlineRadius="5px"
>
    Hello!
</Button1>

// Способ 2. Передача props на стороне стилизуемого компонента
const Button2 = styled.button<FocusProps>`
    ${addFocus({
        outlined: true,
        outlineSize: '2px',
        outlineOffset: '2px',
        outlineColor: 'green',
        outlineRadius: '5px',
    })}
`;

<Button2>Hello!</Button2>
```

## applyDisabled
Делает компонент недоступным для взаимодействия, если компонент может быть задизейблен или если передан флаг ``disabled``.

### Props
<Props of={applyDisabled} />

### Использование
```tsx
import styled from 'styled-components';
import { DisabledProps, applyDisabled } from '@sberdevices/ui/mixins';

const Button = styled.button<DisabledProps>`
    ${applyDisabled}
`;

<Button disabled>I'm disabled</Button>
```

## applyInteraction
Увеличивает/уменьшает компонент при таких видах взаимодействия, как наведение (:hover) и нажатие (:active).

### Props
<Props of={applyInteraction} />

### Использование
```tsx
import styled from 'styled-components';
import { InteractionProps, applyInteraction } from '@sberdevices/ui/mixins';

const Button = styled.button<InteractionProps>`
    ${applyInteraction}
`;

<Button scaleOnInteraction>Button!</Button>
```

## applyRoundness
Применяет к компоненту скругление из списка ``radiuses``.

> Скругление указывается в пикселях x1.
Пример, ``roundness={16}`` = **1rem** = **16px** (при deviceScale x1) = **32px** (при deviceScale x2).

### Props
<Props of={applyRoundness} />

### Использование
```tsx
import styled from 'styled-components';
import { RoundnessProps, applyRoundness } from '@sberdevices/ui/mixins';

const Button = styled.button<RoundnessProps>`
    ${applyRoundness}
`;

<Button roundness={16}>Button!</Button>
```

## applyView
Применяет цвет фона и текста при передаче соответствующего значения в свойство ``view``.

### Props
<Props of={applyView} />

### Использование
```tsx
import styled from 'styled-components';
import { ViewProps, applyView } from '@sberdevices/ui/mixins';

const Button = styled.button<ViewProps>`
    ${applyView}
`;

<Button view="primary">Primary button</Button>
```
