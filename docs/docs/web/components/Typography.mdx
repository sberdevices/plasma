# Typography
В состав Plasma Web входят компоненты, реализующие типографическую систему [типографическую систему Plasma](../../ui/design/typography).
Они заменяют обычные HTML-теги, такие как `h1`, `h2`, `h3`, `h4`, `p` и некоторые другие.

## Установка шрифта
Компоненты типографики используют фирменный шрифт **SB Sans Text**.
О том, как установить данный шрифт, смотрите на странице [Типографическая система](../../ui/design/typography#установка-шрифта).

## Отступы
По умолчанию типографика поддерживает стандартные отступы.
О том, как использовать отступы, читайте на странице [Стандартные отступы](../../ui/design/spacing#встроенные-отступы-в-типографике).

## Состав

### Hero unit
```tsx
<>
    <Display1>Display1</Display1>
    <Display2>Display2</Display2>
    <Display3>Display3</Display3>
</>
```

### Заголовки
```tsx
<>
    <Headline1>Headline1</Headline1>
    <Headline2>Headline2</Headline2>
    <Headline3>Headline3</Headline3>
    <Headline4>Headline4</Headline4>
</>
```

### Основной текст
```tsx
<>
    <Body1>Body1</Body1>
    <Body2>Body2</Body2>
    <Body3>Body3</Body3>
    <ParagraphText1>ParagraphText1</ParagraphText1>
    <ParagraphText2>ParagraphText2</ParagraphText2>
</>
```

### Вспомогательный текст
```tsx
<>
    <Footnote1>Footnote1</Footnote1>
    <Footnote2>Footnote2</Footnote2>
</>
```

### Контролы
```tsx
<>
    <Button1>Button1</Button1>
    <Button2>Button2</Button2>
</>
```

### Дополнительные
```tsx
<>
    <Caption>Caption</Caption>
    <Underline>Underline</Underline>
</>
```
